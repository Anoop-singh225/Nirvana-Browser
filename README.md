### 🚀 Key Highlights
- **🎯 Zero-Distraction Mode:** Custom-built ad-blocking logic to keep you focused on research.
- **⚡ Lightweight Footprint:** Optimized RAM management to run alongside heavy IDEs (VS Code/PyCharm).
- **🛡️ Privacy Sovereign:** No data leaves your machine. Local-only history and cache encryption.
- **📚 Scholar Toolkit:** One-click access to academic repositories and research documentation.

### 🛠️ Installation & Setup
Get the Nirvana Browser up and running in your local environment:

```bash
# Clone the repository
git clone [https://github.com/Anoop-singh225/Nirvana-Browser](https://github.com/Anoop-singh225/Nirvana-Browser)

# Install core dependencies
pip install PyQt5 PyQtWebEngine

# Launch the Engine
python main.py
